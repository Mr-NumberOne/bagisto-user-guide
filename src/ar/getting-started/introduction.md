# مقدمة 

[Bagisto](https://bagisto.com/en/) عبارة عن منصة رائعة للتجارة الإلكترونية مفتوحة المصدر تتيح للشركات إنشاء متاجر حديثة عبر الإنترنت بسهولة. يقدم Bagisto مجموعة واسعة من الميزات القابلة للتخصيص لتلبية الاحتياجات المحددة لأي عمل تجاري عبر الإنترنت.

تم إنشاء النظام الأساسي بناءً على إطار عمل [Laravel](https://laravel.com/) الموثوق به، و[Tailwind CSS](https://tailwindcss.com/) و[Vue.js](https://vuejs.org/)، وهو إطار عمل JavaScript سهل الاستخدام. فهو يوفر للشركات واجهة سهلة الاستخدام وأدوات قوية لإدارة متاجرها عبر الإنترنت والتعامل مع المخزون والطلبات وإنشاء تجربة تسوق سلسة للعملاء.
