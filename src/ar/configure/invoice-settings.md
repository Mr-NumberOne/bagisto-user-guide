# إعدادات الفاتورة

لتهيئة إعدادات الفاتورة، انتقل إلى لوحة الإدارة وانقر على **تكوين >> إعدادات الفاتورة**.

### 1. إعدادات رقم الفاتورة

قم بتكوين القواعد أو المعلمات لإنشاء وتعيين أرقام تعريف فريدة للفواتير لأغراض تنظيمية وتتبعية.

<ImagePopup src="/images/configure/invoicenumberSettings.png" alt="إعدادات رقم الفاتورة" />

### 2. شروط الدفع

وضع شروط متفق عليها تحدد متى وكيف يجب أن يقوم المشتري بدفع ثمن البضائع أو الخدمات إلى البائع.

<ImagePopup src="/images/configure/PaymentTerms.png" alt="شروط الدفع" />

### 3. تصميم قسيمة الفاتورة

قم بتخصيص التصميم المرئي وتنسيق قسيمة الفاتورة، بما في ذلك العلامة التجارية للشركة، والتفصيل، والتسعير، وتفاصيل الدفع للعرض التقديمي الاحترافي.

<ImagePopup src="/images/configure/invoiceSlip.png" alt="تصميم قسيمة الفاتورة" />

### 4. تذكيرات الفاتورة

تمكين الإشعارات أو الاتصالات الآلية المرسلة إلى العملاء لتذكيرهم بدفعات الفواتير القادمة أو المتأخرة.

<ImagePopup src="/images/configure/invoiceReminder.png" alt="تذكير الفاتورة" />

بعد إجراء التغييرات، انقر فوق الزر **حفظ التكوين**.
