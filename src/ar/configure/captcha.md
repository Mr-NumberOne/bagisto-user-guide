# كلمة التحقق

"خريطة الموقع: خريطة تخطيط موقع الويب لمحركات البحث. المفتاح السري: رمز آمن لتشفير البيانات أو المصادقة أو حماية الوصول إلى واجهة برمجة التطبيقات."

في لوحة الإدارة، انتقل إلى **تكوين >> Captcha**، وأضف **مفتاح الموقع والمفتاح السري**، وقم بتمكين/تعطيل **الحالة** وفقًا لمتطلباتك.

<ImagePopup src="/images/configure/captcha.png" alt="Captcha" />

بعد ذلك، انقر على زر **حفظ التكوين**.
