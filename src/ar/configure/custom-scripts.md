# البرامج النصية المخصصة

البرامج النصية المخصصة هي أجزاء مخصصة من التعليمات البرمجية تم إنشاؤها لإضافة وظائف أو ميزات محددة إلى البرنامج، مما يعزز قدراته بشكل فريد.

هنا، في هذا القسم المحدد، يمكن للمسؤول كتابة كود CSS وJS. وفي النهاية، سيتم إلحاق الكود تلقائيًا بالملف الرئيسي.

**خطوات إضافة نصوص برمجية مخصصة**

لإضافة **نصوص برمجية مخصصة**، انتقل إلى لوحة الإدارة: **تكوين >> المحتوى**. ستظهر لك الخيارات التالية:

### CSS مخصص

لقد استخدمنا لون الخلفية **أزرق فاتح** وحفظنا التكوينات.

<ImagePopup src="/images/configure/css.png" alt="CSS مخصص" />

الآن تم تغيير لون خلفية واجهة المتجر، كما هو موضح أدناه:

<ImagePopup src="/images/configure/cssOutput.png" alt="مخرجات CSS المخصصة" />

### JS مخصص

يمكنك تطبيق JavaScript وفقًا لمتطلباتك والنقر فوق الزر **حفظ التكوين**.

<ImagePopup src="/images/configure/js.png" alt="JS مخصص" />

بعد ذلك، انقر على زر **حفظ التكوين**.
