# تكوين أسعار الصرف

تتيح لك أسعار صرف Bagisto تحديث أسعار صرف العملات تلقائيًا باستخدام خدمات API الخارجية.

في لوحة الإدارة، يمكنك إدخال بيانات اعتماد API بسرعة، واختيار مزود الخدمة المفضل لديك، وترتيب جدول زمني للتحديثات التلقائية.

وهذا يضمن صحة أسعار العملات في متجرك دون الحاجة إلى القيام بذلك يدويًا.

لإعداد أسعار الصرف، انتقل إلى: التكوين > عام > أسعار الصرف، كما هو موضح في الصورة أدناه.

  <ImagePopup src="/images/configure/exchange_rates_configurations.png" alt="العنوان" />

### تكوينات أسعار الصرف

فيما يلي الخطوات التفصيلية لتكوينات أسعار الصرف.

**1) الإعدادات العامة**

في هذا الجزء، يمكنك اختيار خدمة سعر الصرف الافتراضية التي سيستخدمها باجستو.

يمكنك الاختيار بين واجهة برمجة تطبيقات أسعار الصرف وواجهة برمجة تطبيقات Fixer، بناءً على ما تفضله وما تحتاجه.

**2) واجهة برمجة تطبيقات المثبت**

هنا، يجب عليك إدخال مفتاح API الذي حصلت عليه منfixer.io.

يمنحك Fixer معلومات في الوقت الفعلي ومعلومات عن أسعار الصرف السابقة. كما أن لديها خطة مجانية، ولكن يمكنك فقط تقديم عدد محدود من طلبات API.

**3) واجهة برمجة تطبيقات أسعار الصرف**

في هذا القسم، تحتاج إلى إدخال مفتاح API الخاص بك من Exchangerate-api.com. توفر هذه الخدمة أسعار صرف فورية لأكثر من 160 عملة.

تتيح لك الخطة المجانية إجراء ما يصل إلى 1500 طلب شهريًا، وهو أمر جيد لمعظم المتاجر الصغيرة والمتوسطة الحجم.

**4) الاستيراد المجدول**

يمكنك تشغيل التحديثات المجدولة لتحديث أسعار الصرف تلقائيًا في أوقات محددة.

بمجرد تشغيله، يمكنك اختيار عدد مرات التحديث - يوميًا أو أسبوعيًا (كل يوم اثنين) أو شهريًا (في اليوم الأول من الشهر).

يمكنك بسهولة إضافة هذا من الإعدادات كما هو موضح أدناه.

<ImagePopup src="/images/configure/schedule.png" alt="العنوان" />

يجب عليك ضبط وقت البدء باستخدام **تنسيق 24 ساعة**، مثل هذا: 03:00.

للتأكد من أن التحديثات المجدولة تعمل بشكل صحيح، تحتاج إلى تشغيل برنامج جدولة المهام Laravel.

يمكنك البدء باستخدام هذا الأمر: 

** جدول الحرفيين PHP: العمل **

### النسخ الاحتياطي لمتغيرات البيئة

إذا نسيت وضع مفاتيح API في لوحة الإدارة، فسيحصل Bagisto تلقائيًا على مفاتيح API من ملف .env الخاص بك.

بالنسبة إلى Fixer API، يمكنك استخدام **FIXER_API_KEY**.

بالنسبة لواجهة برمجة تطبيقات أسعار الصرف، يمكنك استخدام **EXCHANGE_RATES_API_KEY**.

**مثال لتكوين .env:**

FIXER_API_KEY=your_fixer_api_key_here

EXCHANGE_RATES_API_KEY=your_exchange_rates_api_key_here

### التحديث اليدوي

إلى جانب التحديثات التلقائية، يمكنك أيضًا تغيير أسعار الصرف يدويًا.

للقيام بذلك، انتقل إلى لوحة الإدارة، ثم انقر فوق **الإعدادات > أسعار الصرف** واضغط على زر **تحديث الأسعار**.

وبدلاً من ذلك، يمكنك استخدام الأمر في واجهة سطر الأوامر: **php artisan Exchange-rate:update**

### واجهة برمجة تطبيقات أسعار الصرف (exchangerate-api.com):

- مثال لنقطة النهاية: **v6.exchangerate-api.com/v6/{API_KEY}/latest/{BASE}**

- جلب جميع أسعار العملات في مكالمة API واحدة

- الطبقة المجانية: 1,500 طلب/الشهر

- الاشتراك: [https://www.exchangerate-api.com](https://www.exchangerate-api.com)

### الخدمات المدعومة

واجهة برمجة تطبيقات أسعار صرف باجيستو (exchangerate-api.com)

تحصل هذه الأداة على جميع أسعار صرف العملات من خلال استدعاء API واحد فقط. يمكنك تقديم 1500 طلب مجاني كل شهر.

### واجهة برمجة تطبيقات المثبت (fixer.io)

- نقطة النهاية: http://data.fixer.io/api/{date}

- جلب الأسعار لكل زوج من العملات

- الطبقة المجانية: 100 طلب/شهر

- الاشتراك: [https://fixer.io](https://fixer.io)
